Task 4: Exploitation & System Security
Objective
The objective of this task was to understand the penetration testing workflow, exploit vulnerabilities in a controlled environment, analyze security weaknesses, and implement defensive security measures.

Tools Used
Kali Linux
Metasploit Framework
Hydra
John the Ripper
Wireshark
Linux Terminal
Metasploitable2
VirtualBox / VMware
Activities Performed
1. Penetration Testing Methodology
Studied the complete penetration testing lifecycle:
Reconnaissance
Scanning
Exploitation
Post-Exploitation
Reporting
Documented each phase with observations and findings.
2. Exploitation with Metasploit
Identified vulnerable services on Metasploitable2.
Exploited known vulnerabilities using Metasploit Framework.
Gained controlled access to the target machine.
Performed post-exploitation enumeration and system analysis.
3. Password Security Assessment
Studied password attack methodologies.
Performed password auditing in a controlled lab environment.
Analyzed password hashes using John the Ripper.
Learned authentication security best practices.
4. Social Engineering Awareness
Studied common social engineering techniques.
Created a phishing awareness simulation for educational purposes.
Identified indicators of phishing attacks.
Reviewed user awareness and prevention strategies.
5. Malware Basics
Studied malware categories and attack vectors.
Understood Static Analysis and Dynamic Analysis concepts.
Observed malware behavior in a sandbox environment.
Learned about Indicators of Compromise (IOCs).
6. System Hardening
Applied security patches and updates.
Configured firewall rules to reduce attack surface.
Disabled unnecessary services.
Implemented basic system security best practices.
